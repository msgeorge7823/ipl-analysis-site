Data accuracy is the #1 priority. The user has emphasized this is a professional, authentic application. Wrong data destroys credibility.

**Why:** This is meant to be a broadcast-grade professional analytics platform. Every data point must be verifiable.

**How to apply:**
- NEVER guess or approximate player data (roles, teams, status, names)
- Always verify against official sources: iplt20.com, ESPNCricinfo, CricBuzz
- Player roles must come from official team sheets, not inference from stats
- Squad compositions must match official retention/auction announcements
- Retirement status must be from official announcements only
- When in doubt, search online to verify before displaying
- Cross-reference multiple sources for critical data points
- Keep data current as of today's date
